# autohotkey-scripts
my autohotkey scripts
<br/><br/>
i3.ahk -common keyboard shortcuts mostly inspired by i3 in linux<br/>
  pressing ltrl sends ctrl, tapping sends esc<br/>
  (this is used with a registry mod that sets capslock to lctrl)<br/>
  win+shift+q closes current application<br/>
  win+enter launches terminal<br/>
  common software shortcuts<br/>
<br/>
mouse.ahk -adds keyboard layers, and options to emulate numpad and mouse<br/>
  alt+space cycles layer (for now, find better shortcut)<br/>
  layer 0: normal mode<br/>
  layer 1: wasd moves mouse, q and e are left and right click, ctrl+wasd moves faster, shift+wasd slower.  ctrl+q double-click, shift+q toggle click for dragging<br/>
  layer 2: numpad (123 456 789) (m,. jkl uio)<br/>
